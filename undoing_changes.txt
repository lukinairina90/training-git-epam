Что делать если ты наговнячил и хочешь откатиться
Звездочки (*) - это примечания

1.(*) Изменения ещё не добавлены в инднкс(не было git add)
(WORKING DIRECTORY)
    git checkout -- file.txt
    git checkout .
    git clean -xdf

2.(**) Изменения в индексе (сделали git add)
(INDEX STAGING AREA)
    git reset -- file.txt

3.(***) Изменения закомичены (сделали git commit)
(LOCAL BRANCH)
    git reset HEAD^^ (HEAD~2)
    git commit --amend -m "commit message"

4.(***) Изменения запушины (git push)
(REMOTE REPOSITORY)
    git revert <sha1>

*   git checkout .: точка означает, что откатываем изменения во всех файлах
    git clean -xdf: удаляем новые созданные файлы, х - игнорим гитигнор и всякие важные файлы, d - удаляем папки,
                    f - форс(хрен знает, что это значит)
