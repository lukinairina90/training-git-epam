Что делать если ты наговнячил и хочешь откатиться
Звездочки (*) - это примечания
________________________________________________________________________________________________________________________

1.(*) Изменения ещё не добавлены в инднкс(не было git add)
(WORKING DIRECTORY)
    git checkout -- file.txt
    git checkout .
    git clean -xdf
________________________________________________________________________________________________________________________
2.(**) Изменения в индексе (сделали git add)
(INDEX STAGING AREA)
    git reset -- file.txt
________________________________________________________________________________________________________________________

2.(**) Изменения в индексе (сделали git add)
(INDEX STAGING AREA)
    git reset -- file.txt

3.(***) Изменения закомичены (сделали git commit)
(LOCAL BRANCH)
    git reset HEAD^^ (HEAD~2)
    git commit --amend -m "commit message"
________________________________________________________________________________________________________________________

4.(***) Изменения запушины (git push)
(REMOTE REPOSITORY)
    git revert <sha1>
________________________________________________________________________________________________________________________

________________________________________________________________________________________________________________________
*   git checkout .: точка означает, что откатываем изменения во всех файлах
    git clean -xdf: удаляем новые созданные файлы, х - игнорим гитигнор и всякие важные файлы, d - удаляем папки,
                    f - форс(хрен знает, что это значит)

**
*** git commit --amend -m "commit message": дополняем коммит(не удаляя его)
    git reset HEAD^^ (HEAD~2) - откатиться на сколько то там коммитов пишем или: git reset HEAD^^ (на 2 коммита, ^(один))
                                или так git reset HEAD~2: ~2(выбираем на сколько откатиться)
**** git revert <sha1>: sha1 - это первые пять цифр в огромной буквенно-цифровой строке, которую можно увидеть после комманды
                        git log (commit 96f6747f6fb23ae12a3d39e0d1141d65528c6fa2), sha1 это 96f67
